Proyecto: Fecha limite 12-21 de Noviembre del 2024

TEMA: Web sobre destinos turisticos de todo el mundo con INICIO DE SESION
Contenidos:
	Landing page
	Registro e Inicio de sesion
	Dashboard para crear y subir destinos turisticos
	Blog para visualizar los destinos turisticos creados por todos los 
	usuarios

Funcionalidades:
	Iniciar sesion
	Registrarse
	Crear destinos
	Borrar destinos
	Actualizar destinos
	Ver todos mis destinos
	Ver los destinos de los destinos de los demas
	Cerrar sesion

Reglas: 
 No está permitido el uso de recursos
 externos para la aplicación de estilos
 (Bootstrap, Tailwind, etc.).
 Se permiten librerías que sean para
 colocar animaciones en la página.
 Se debe levar un control de versiones con
 GIT y se deben realizar mínimo 15 commits.
 Subir a repositorio de Github. 
Se debe desplegar el resultado en Github
 Pages.
 Debe ser responsive
