instrucciones proyecto:
	Fecha limite: 12-21 Noviembre
	Tematica: Blog de Turismo
	Objetivo: Informar sobre destinos turisticos en el mundo
	
Contenidos:
	Landing page
	Registro e Inicio de sesion
	Dashboard para crear y subir destinos turisticos
	Blog para visualizar los destinos turisticos creados por los usuarios

Fucionalidades:
	Iniciar sesión.
	Registrarse.
	Crear destinos.
	Borrar destinos.
	Actualizar destinos.
	Ver todos mis destinos.
	Ver los destinos de los demás usuarios.
	Cerrar sesión.

Reglas:
	No está permitido el uso de recursos
externos para la aplicación de estilos (Bootstrap, Tailwind, etc.).
	Se permiten librerías que sean para
colocar animaciones en la página.
	Se debe llevar un control de versiones con
GIT y se deben realizar mínimo 15 commits.
	Subir a repositorio de Github.
	Se debe desplegar el resultado en Github
Pages.
	Debe ser responsive.
